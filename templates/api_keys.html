ℹ️ About API Keys

API keys allow you to authenticate API requests without using a session cookie. This is useful for:

  • Programmatic access from scripts or automation tools
  • Integration with external systems
  • Command-line tools like curl

Usage: Include your API key in the X-API-Key header:

curl -H "X-API-Key: your-api-key-here" https://your-domain.com/api/services/1/status
